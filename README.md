项目简介：
     这是一个基于 RAG (Retrieval-Augmented Generation) 架构的 AI 助手，专门为算法竞赛 (CCPC/ICPC) 学习者设计。它通过检索本地全量 OI-Wiki 文档，结合 DeepSeek 的大脑，提供专业、准确的算法指导。

技术栈：
- **框架**: LangChain
- **模型**: DeepSeek-V3 (LLM) + BAAI/bge-small-zh-v1.5 (Embedding)
- **数据库**: ChromaDB (Vector Store)
- **数据源**: OI-Wiki (Markdown)

核心功能
- **全量知识覆盖**：自动索引 OI-Wiki 超过 400 个算法专题，近万个知识块。
- **精准语义检索**：采用 BGE (Beijing General Embedding) 模型，理解中英文算法术语。
- **开卷考试模式**：AI 回答时严格参考本地资料，拒绝“幻觉”，并支持信源溯源。
- **自动审计代码**：ai根据标准答案检查你提供的代码
